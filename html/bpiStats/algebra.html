Algebra

Writing equations

Algebra is a big subject but we really only need to focus on one thing: How to solve for a variable in a linear equation. Take for example the puzzle:

"I have a number which when you add 2 you get 7. What is my number?"

Obviously the answer is 5 but I chose a really easy example to help you see what Algebra is. This question is the same as the Algebra problem $$ x+2 = 7 $$

You could run into a real-life problem like this if, say, you and a friend are paying for lunch. The bill is $25 and your friend pays $15, and you need to cover the rest. Again the numbers are simple but it's just to keep the picture clear so that you can see the connection to the equation $$x + 15 = 25$$

Here's another puzzle that's harder to solve without algebra.

"8 years ago, one third of my age was equal to a fifth of how old I'll be 5 years from now. How old am I?"

You might still be able to get through this without algebra but I think algebra is just easier. First, since we don't know the age in question, we just write it as x in order to be able to talk about it. Now "8 years ago" is talking about when I was x-8. So we take a third of that, which can be written as \(\frac{x-8}{3}\).

Now that's equal to my age in five years, divided by 5. We can write that as \(\frac{x+5}{5}\). Since these two quantites are said to be equal, we write an equation.

$$\frac{x-8}{3}=\frac{x+5}{5}$$

Test your understanding

Write an equation corresponding to the following sentence:

"If you add three books to my bookshelf, there will be just as many books as if you removed half and added 10."

Answer

$$x+3 = \frac{x}{2}+10$$

How to solve equations (or: the art of cancelling)

Now that we've seen how equations correspond to real world questions, let's talk about how to solve them. We're going to look at how to solve these 10 equations:

1.$$x+2=7$$
2. $$2x = 10$$
3. $$2x-7=11$$
4. $$\frac{x/2-3}{4}=5$$
5. $$2x-3x+1=-10$$
6. $$ 5(x-1)-2(x+1) = 10 $$
7. $$2(3x-1) = 22$$
8. $$ 2(3x-1) = 9x $$
9. $$\frac{1}{x}+1 = 2$$
10. $$\frac{2x-3}{x} = 5$$

In every case we're going to rely on the idea of canceling! We obviously don't need it for the first two because the solution is so easy, but let's start warming up to the idea of canceling.

Problem 1.   \(x+2=7\)

The first principle to make clear is that whatever you do to one side you must do to the other. So if I wanted to I could add 1 to both side. It's not a good idea to do this, but you could. It won't get you any closer to the answer. However, if I add -2 to both sides then I will cancel the 2 on the left and get rid of 2.

$$x+2-2=7-2$$ therefore $$x=5$$ On the left it cancels, on the right it computes. We'll see this pattern over and over again.

Problem 2.   \(2x = 10\)

This time the idea of cancelling is a little bit different because we don't add anything. We want to cancel the 2, but this time it's multiplied. The thing that cancels multiplication is division, so we divide by 2.

$$2x\div 2 = 10\div 2$$ therefore $$x = 5$$ Again it cancelled on the left and computed on the right.

Problem 3.   \(2x-7=11\)

This time we have a question: Which do we cancel first? Do we cancel the -7 or the 2*? If we try to cancel the 2* we should realize that this is a mistake. Why?

It's very important to understand the reason: Because if you divide by 2 you have to divide the ENTIRE left side by 2. But that doesn't just cancel because the -7 is also getting divided. If you tried to do this it should look like this:

$$\frac{2x-7}{2}=\frac{11}{2}$$

Now this is not actually wrong. You could get to the answer from here. But it's harder than it has to be. If we cancel the -7 first, that will be simpler because we can immediately cancel it.

$$ 2x-7+7 = 11+7 $$ therefore $$ 2x = 18 $$

Notice that it cancelled on the left and computed on the right. Now we can divide by 2 to get the answer

$$\frac{2x}{2}=\frac{18}{2}$$ therefore $$x = 9$$

Problem 4.   \(\frac{x/2-3}{4} = 5\)

Now we have three things to choose from. Do we cancel the /2, do we cancel the -3, or do we cancel the division by 4? The answer is that we look for the "outer-most operation". In this case that's the division by 4. In some sense it's the "main operation" because it's dividing everything else by 4. So we start with that.

To cancel it we multiply by 4.

$$4\times \frac{x/2-3}{4}=5\times 4$$ therefore $$ x/2-3 = 20$$

Now we have to identify the "main operation" again, and this time it's the -3 because everything else is being subtracted by 3. So to invert it we add 3.

$$ x/2-3+3 = 20+3 $$ therefore $$ x/2 = 23 $$

Finally there's only one operation left so we invert it by multiplying by 2.

$$2\times x/2 = 23\times 2$$ therefore $$x = 46$$

Problem 5.   \(2x-3x+1=-10\)

This one is different. The only important difference is that there isn't just one x in the equation. We don't have techniques for solving when there are xs located in multiple places. Therefore we have to make it so that there is just one x!

We can do that by factoring out the x.

$$x(2-3)+1=-10$$ therefore $$x(-1)+1=-10$$

Now we're in a situation very similar to Problem 3. We solve it the same way, starting by cancelling the +1 and then cancelling the *(-1).

$$x(-1)+1-1=-10-1$$ therefore $$x(-1) = -11$$ therefore $$\frac{x(-1)}{-1}=\frac{-11}{-1}$$ therefore $$x = 11$$

Problem 6.   \(5(x-1)-2(x+1)=10\)

Again there are many xs so we have to somehow put them together. However, this time they each occur inside their own multiplication term. Therefore we have to "break them out". In this case that means "distributing" and then grouping them together.

$$ 5x-5-2x-2 = 10$$

Now that the two terms have been distributed into 4 terms, the terms with xs are free to be grouped together.

$$5x-2x-5-2 = 10$$ therefore $$x(5-2)-7=10$$ therefore $$x\times 3-7 = 10$$ therefore $$x\times 3-7+7=10+7$$ therefore $$x\times 3 = 17$$ therefore $$x\times 3/3 = 17/3$$ therefore $$x = 17/3$$

Problem 7.   \(2(3x-1)=22\)

This time there's only one x so we don't have to distribute or factor. We can just cancel the main operation. I'll stop writing "therefore" every time and I'll stop showing all of the steps of calculation. You should be able to see which steps I'm taking.

$$2(3x-1)\div 2 = 22\div 2$$
$$3x-1 = 11$$
$$3x = 12$$
$$x = 4$$

Problem 8.   \(2(3x-1) = 9x\)

Again we have xs in many places. This time it's a little different from the other examples because an x is on both sides of the equal-sign. So we can't just distribute and move them together. We need to somehow take one of them and move it to the other side of the equal-sign. That's simple enough, though: Cancelling by -9x will do it.

$$2(3x-1)-9x=9x-9x$$
$$2(3x-1)-9x = 0$$

This problem is pretty similar to problem 6 now. We distribute, group, and cancel. $$6x-2-9x = 0$$
$$x(6-9)-2=0$$
$$x(-3)-2=0$$
$$x(-3)=2$$
$$x = 2/(-3)$$
$$x=-2/3$$

Problem 9.   \(\frac{1}{x} + 1 = 2\)

In this problem we have something significantly different from the others. The x is in the bottom of a fraction. Since that is such a problem, we need to move the x out of the bottom. Yet again cancelling can do this for us. If we want to cancel the division by x we can do so by multiplying by x.

$$x\times \left(\frac{1}{x}+1\right) = x\times 2$$
$$x\times \frac{1}{x} + x = 2x$$
$$1 + x = 2x$$

Notice the cancellation of the xs in \(x\times \frac{1}{x}\) which became just 1.

Now that it's in this form it looks a lot like previous problems. We group the xs together, factor, cancel.

$$1=2x-x$$ $$1=(2-1)x$$ $$1=x$$

Problem 10.   \(\frac{2x-3}{x}=5\)

$$ x\times \frac{2x-3}{5}=5\times x $$
$$2x-3 = 5x$$
$$-3 = 5x-2x$$
$$-3 = (5-2)x$$
$$-3 = 3x$$
$$-1 = x$$

Test you understanding

Solve $$\frac{x}{2}-\frac{x}{3}=1$$

Answer

Of course we must group xs and factor them out so that there's only 1 x. We can do that immediately or we can add fractions first and then do that. I'll just do it immediately.

$$x\left(\frac{1}{2}-\frac{1}{3}\right) = 1$$ $$x\left(\frac{3}{6}-\frac{2}{6}\right) = 1$$ $$x \times (1/6) = 1$$ $$x = 6$$

Ideas needed to solve equations

To recap, all of solving algebra problems is about cancelling. But to cancel you need to have just a single x in the equation. If you don't then you're going to need to use factoring and distributing so that you make there be only one x. It's important to remember when factoring that, if there isn't a coefficient then factoring out x leaves a 1. For example if we factor \(5x-x+2x+x\) this is the same as \(x(5-1+2+1)\). Once you've made it so that an equation just has a single x then you need to start cancelling the operations, starting with the "main operator".

Solving "for a variable" or "in terms of a variable"

One last thing to consider is how to solve "for a variable". It will often be the case that an equation doesn't just have an x in it, but it also has a bunch of other variables. A very, very frequently occurring equation in statistics is $$z = \frac{x-\mu}{\sigma}$$

This equation is already "solved for z". So if you need to solve for z ... well, there's just nothing to do. It's already done.

But if you need to solve for x then you use all of the same ideas as before. Except this time there are no numbers, just variables. Which is great because computing numbers is boring. So here's how you'd solve for x.

$$\sigma\times z = x-\mu$$ $$\sigma z + \mu = x$$

Test your understanding

Solve the equation \(z = \frac{x-\mu}{\sigma}\) for \(\sigma\).

Answer

$$\sigma z = x-\mu$$ $$\sigma = \frac{x-\mu}{z}$$

Test your understanding

Suppose that 60% of people at a party are males and you know that the number of females is 10. How many people are at the party?

Answer

Clearly the percent of females is 40% and it will be better to relate the percent of females to the number of females. The unknown quantity is the total number of people, so call it x. We know that if you multiply a total times a percent you get the amount of the part. So we can say $$40% \times x = 10$$ which is the same as $$0.4 x = 10$$ and then you can cancel the 0.4 by division. $$x = 10/0.4 = 100/4 = 25$$
