Arithmetic

We're going to obviously do a lot of calculations in this class so let's make sure you recall the fundamentals, boring as it might be.

Adding all kinds of *&^#

Of course you all recall how to add 123+456 kind of stuff. How about 1.23 + 45.6? The key is to line up the decimal point and add like usual. Same thing goes for subtraction. You should test yourself to make sure you got this. If you do it right, you should get that 1.23+45.6 is 46.83.

Now how about adding negatives? You might have forgotten this depending on how long ago your math class was and how far you went. What is -12+3? Step one is to determine what the sign is. The negative is "bigger" than the positive so the answer is going to be negative. Next, because one is positive and the other is negative they're going to cancel, not combine. Therefore we'll take the difference of 12 and 3 which is 9. Put it all together now and the result is -9.

Test your understanding

Compute the following.

  1. 1.23-4.56
  2. -1.23-4.56
  3. -1.23-(-4.56)

Answers

  1. First notice the negative is larger (it's the -4.56 because subtraction is just like adding with a negative). Since the signs are opposite we're going to cancel so we subtract. It'll be easier to do if we put the larger number first.

    $$\begin{array}{c c c c} & 4. & 5 & 6\\ -& 1. & 2 & 3 \\\hline & 3. & 3 & 3 \end{array}$$

    So the solution is -3.33.

  2. Since both numbers are negative the result is just going to be a negative number which combines them. So we might as well just add $$\begin{array}{c c c c} & 4. & 5 & 6\\ +& 1. & 2 & 3 \\\hline & 5. & 7 & 9 \end{array}$$ and so the answer is -5.79.

  3. Here you have to recall that subtracting a negative just list adding. The negatives cancel, so that this is the same thing as -1.23+4.56. Now the bigger number is positive so the answer will be positive. But since the signs are different we have to cancel. Therefore we perform the subtraction just like in the first exercise, and it comes out to 3.33. Since the answer stays positive this time, this is the answer.

Now recall how to add fractions. If the denominator is the same the process is easy: Just keep the denominator and add the numerator. For instance 1/2 + 1/2 is 2/2. And of course we can simplify that as just 1. Subtraction works the same way.

What about adding when the denominators are different? How do you add 1/2 + 1/3? The key is to split the 1/2 into three pieces and split the 1/3 in two pieces. 1/2 divided into 3 pieces gives you 3 pieces each with size 1/6, so it's the same as 3/6. 1/3 split in 2 pieces gives you 2 pieces each with size 1/6 so it's 2/6. So now we know that 1/2 + 1/3 is the same as 3/6 + 2/6 and we can use the old rule for adding fractions with the same denominator. It's 5/6.

Multiplication

You probably recall how to multiply large numbers like 12*45 but just to be sure, recall that you multiply the first number by each digit of the second number, but as you go you offset the digits.

$$\begin{array}{c c c c} & & 1 & 2\\ \times& & 4 & 5 \\\hline & & 6 & 0 \\ & 4 & 8 & 0 \\ \end{array}$$

Notice that you have to do some "carrying" to get this result, but I don't show that since it can confuse the picture. Then you take these rows and add them.

$$\begin{array}{c c c c} & & 1 & 2\\ \times& & 4 & 5 \\\hline & & 6 & 0 \\ +& 4 & 8 & 0 \\\hline & 5 & 4 & 0 \end{array}$$

Ok, but how do you multiply decimals? Here's how I do it: Multiply them like there's no decimal at all. Then put the decimals back in at the end. So if you multiply 1.2*4.5 I would do it like this: Multiply 12*45 and get 540. Now the first one had a decimal so you change it to 54.0. The second one also had a decimal so you change it to 5.4. Now you have the answer.

Test your understanding

Find 12.5 * 1.01.

Answer

If we multiply like there's no decimals we find 125*101. You can use the table method you see above, or you can do a little trick. 125*101 is almost the same thing as 125*100 and that's 12500. To get the last multiple of 1 just add on 125 to this. Now you get 12625.

Ok, now that we know 125*101 is 12625, we put the decimals back in. The first number had one decimal so we change it to 1262.5. The second number had two decimals so we change the answer to 12.625 and we're done.

Getting back to different kinds of multiplication, how do you multiply negatives? What's 2*(-3)?

  1. If the numbers have the same sign, the multiplication is positive.
  2. If the numbers have different signs, the result is negative.
  3. In both cases, you'll still just multiply the numbers without signs.
Using that rule, we know the result will be negative. So we just multiply 2*3 and get 6. So the answer is -6. These rules should make it clear that -2*(-3) is 6.

And finally, how do you multiply fractions? What is 1/2 * 1/3? The answer is you just multiply tops with tops and bottoms with bottoms. So to get the top we do 1*1 which is 1. To get the bottom we do 2*3 which is 6. The answer is 1/6.

Answer

Division

For some numbers it's obvious you can't do any really good division. Like what's 1 divided by 2? Well really all you can say is it's 1/2 which isn't much better than just saying "1 divided by 2". But for other numbers division is more useful, like 24/6 = 4. For big numbers, we need a method to help us keep track of what we're doing. For example if we want to calculate 12625/125 we need to find the biggest number we can multiply by 125 and stay under 12625. We can start by writing the numbers like this: $$ \require{enclose} \begin{array}{rll} 125 \enclose{longdiv}{12625}\kern-.2ex \end{array} $$ and we start by trying to fit 125 into 126 (we can't hope to fit it in 1 or 12, so we start with 126). It goes in once, so we write $$ \require{enclose} \begin{array}{rll} 1\ \ \ \ && \\[-3pt] 125 \enclose{longdiv}{12625}\kern-.2ex \end{array} $$ then multiply \(125\times 1\) and get 125, put that underneath, subtract. $$ \require{enclose} \begin{array}{rll} 1\phantom{00}&& \\[-3pt] 125 \enclose{longdiv}{12625}\kern-.2ex \\[-3pt] \underline{-125\phantom{00}} && \\ 1\phantom{00} \end{array} $$ next carry a digit from the 12625 $$ \require{enclose} \begin{array}{rll} 1\phantom{00}&& \\[-3pt] 125 \enclose{longdiv}{12625}\kern-.2ex \\[-3pt] \underline{-125\phantom{00}} && \\ 12\phantom{20} \end{array} $$ Since this "goes in" 0 times, we record the 0 and bring down another digit. $$ \require{enclose} \begin{array}{rll} 10\phantom{0}&& \\[-3pt] 125 \enclose{longdiv}{12625}\kern-.2ex \\[-3pt] \underline{-125\phantom{00}} && \\ 125 \end{array} $$ This goes in 1 time so we record it $$ \require{enclose} \begin{array}{rll} 101&& \\[-3pt] 125 \enclose{longdiv}{12625}\kern-.2ex \\[-3pt] \underline{-125\phantom{00}} && \\ 125 \end{array} $$ multiply 125*1 and subtract. $$ \require{enclose} \begin{array}{rll} 101&& \\[-3pt] 125 \enclose{longdiv}{12625}\kern-.2ex \\[-3pt] \underline{-125\phantom{00}} && \\ 125 && \\ \underline{-125} && \\ 0 && \end{array} $$ And since we've ended with a 0 the division is done. What this shows is that $$12625/125 = 101$$ Of course equivalent to this fact is the fact that $$12625 = 125\times 101$$ which is the same thing we saw in the section on multiplication.

Division of decimals is like multiplication with decimals: Do it without the decimals and add them back in. The only difference is that you add back in the decimals that were in the numerator. You subtract out decimals that were in the denominator. So if you had to divide 126.25/12.5 the result would be 101 if we ignore the decimals. Then we add in the two decimals from the numerator to get 1.01. But we move the decimal in the other direction because of the one decimal in 12.5. That means we end with 10.1.

Test your understanding

  1. What is 0.002/200?
  2. What is 5/0.1?

Answers

  1. If you ignore the decimal we're basically calculating 2/200. If you compute this like we did above, you should get $$\require{enclose} \begin{array}{rll} 0.01 && \\ 200 \enclose{longdiv}{2.00} \end{array} $$ This is telling us that 2/200 is 0.01. But because of the three decimals that the original numerator had, we move the decimal left three times. That gives us 0.00001.
  2. For this we ignore decimals and compute 5/1 we already know that comes out to 5. But because of the decimal in the bottom we move the decimal place right by 1. So we get 50.

The rule for negatives is just like the one for multiplication. If you have two negatives or two positives the result is positive. If you have one negative, one positive, the result is negative. But otherwise you just do the division while ignoring the sign.

For fractions we use the "flip and multiply rule". How does that go again? How do you compute, for example, \(\frac{1/2}{3}\)? Do you flip the 1/2?

NO. The rule doesn't have the best name since it doesn't tell you what to flip. You flip the denominator. In this case that's 3, which again we have to write as 3/1 before we can apply the rule. But now if we write it as \(\frac{1/2}{3/1}\) then to flip (the denominator) and multiply we do $$\frac 1 2 \times \frac 1 3$$ How do we finish the calculation? Recall how to multiply fractions. Multipy tops with tops, bottoms with bottoms. Then you get the answer, 1/6.

Exponents, roots, logs

Just like multiplication means repeated addition, exponentiation means repeated multiplication. So 3*4 means 3+3+3+3 which is 12. Or does it mean 4+4+4? Well since that's also 12 it doesn't matter.

Anyway, 34 means 3*3*3*3 which is 81. Or is it 4*4*4 which is 64? This time they're different so we better make sure we get it right. 34 is in fact the first one, so it's 81.

Now every operation we've seen so far has an "inverse". For addition the inverse is subtraction. For multiplication the inverse is division. What's the inverse for exponentiation? Well we have to get clear about what exactly an inverse even means. With addition an inverse looks like this $$ 3+4-4 = 3 $$ which says that if you add 4 and then subtract 4 your right back to where you started. The same idea is true for division. $$ 3\times 4 \div 4 = 3 $$ (In fact this is basically why you can't divide by 0. It's related to the fact that ANY number times 0 is 0.) Notice that because + and ÷ have a symmetry to them, it doesn't matter what order we do it in. $$4+3-4 = 3$$ $$4\times 3\div 4 = 3$$ But that's not true with exponents. We already saw that the order matters. What that means is that there are TWO kinds of inverses for exponents. The first one is what we call the "root". It cancels the exponent and leaves the base. The most famous root is the square root, which cancels a square. $$\sqrt{4^2} = 4$$ In effect this shows you that \(\sqrt{16}=4\).

Test your understanding

Show that \(\sqrt{25}=5\).

Answer

This is because $$\sqrt{25} = \sqrt{5^2} = 5$$

Now getting back to the lesson, is should be clear that \(\sqrt{1}=1\) and \(\sqrt{0}=0\) which are often useful to know because they're so easy to calculate.

But the square root isn't the only root. There's also the cube-root which cancels cubes. Since \(\sqrt[3]{2^3} = 2\) this shows us that \(\sqrt[3]{8}=2\).
