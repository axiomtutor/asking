Chapter 2

Subgroups

2.1 Definition and Examples

A subgroup is a subset that's a group. An equivalent condition is that it's a nonempty subset that has every \(xy^{-1}\).

2.2 Centralizers and Normalizers, Stabilizers and Kernels

The centralizer \(C_G(A)\) is the set of all group elements that fix each of A under conjugation. It is a subgroup of G. The center \(Z(G)\) is the set of elements that commute with everything. The normalizer \(N_G(A)\) is the set of elements that fix ALL of A under conjugation.
