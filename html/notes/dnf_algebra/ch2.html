Chapter 2

Subgroups

2.1 Definition and Examples

A subgroup is a subset that's a group. An equivalent condition is that it's a nonempty subset which has every \(xy^{-1}\). If H is a subgroup of G we write \(H\leq G\).

2.2 Centralizers and Normalizers, Stabilizers and Kernels

The centralizer \(C_G(A)\) is the set of all group elements that fix each of A under conjugation. It is a subgroup of G. The center \(Z(G)\) is the set of elements that commute with everything. The normalizer \(N_G(A)\) is the set of elements that fix ALL of A under conjugation.

Note that these are pure group concepts and need not reference actions.

Stabilizers and Kernels of Group Actions

The stabilizer of a G-set element s is \(G_s\), the set of all group elements which fix s. The fact that centralizers, normalizers, and centers are subgroups are all special cases of the fact that stabilizers are subgroups. For cecntralizers, use the group action of conjugation; for the center, notice that this is the kernel of the same action. For normalizers use the action of conjugation on subsets of G.

2.3 Cyclic Groups and Cyclic Subgroups

A group is cyclic if it's generated by a single element.

Theorem: If \(x^a = \mathbb 1\) then \(|x|\) divides a.

If \(x^m=\mathbb 1 = x^n\) then if \(d=(m,n)\) then we have $$ x^d=\mathbb 1 $$

In particular if \(x^m=\mathbb 1\) then the order of x divides m. The proof goes by Bezout's identity.

Theorem: Cyclic groups of the same order are isomorphic.

Map generator to generator and let the homomorphism guide the rest.

Theorem: The order of a power of a generator is the order over the gcd.

$$ |x^a| = \frac{|x|}{(|x|,a)} $$ To see this write |x| and a as mutliples of d. $$ |x| = db, \quad a = dc $$

With this notation we want to show \(|x^a| = b\). We can show \(|x^a|\) divides b just by taking a power of b. To show that b divides \(|x^a|\) we think about the fact that \((x^a)^{|x^a|} = \mathbb 1\). This entails that \(|x|\) divides \(a|x^a|\). With a little more unraveling and a number-theoretic proof that \((b,c) = 1\) you find that \(|b|\) divides \(|x^a|\).

Theorem: If \(G = \langle x\rangle\) then also \(G = \langle x^a\rangle\) if and only if a and n are coprime. In particular the number of generators is \(\varphi(n)\) where n is the order and \(\varphi\) is Euler's totient function.

The argument goes entirely by the order of the subgroup generated by \(x^a\). It's everything if and only if it's order is the order of the group. By the previous theorem, that happens precisely when the order and power are coprime.

Theorem: Subgroups of cyclic groups are cyclic.

Take the smallest positive d such that \(x^d\) is in the group. Argue that this generates the group by exhibiting any other element as some \(x^a\) and then decomposing into quotients and remainders. Minimality implies the remainder must be 0.

Theorem: For cyclic groups, there is a unique subgroup for every divisor of the order.

Show that the order of \(x^{n/a}\) witnesses the claim.

Theorem: If n is the order of a cyclic group then \(\langle x^m\rangle = \langle x^{(n,m)}\rangle\).

Certainly \(\langle x^m\rangle \leq \langle x^{(n,m)}\rangle \). From the formula for orders of powers of generators, they have the same size.

2.4 Subgroups Generated by Subsets

The smallest subgroup which contains a set \(A\subseteq G\) is the intersection over all subgroups which contain A. It is easy to show this is a subgroup. It is also called the subgroup generated by A.

This is the top-down approach; we can arrive at the same subgroup by closing the set under the operation and inverses.

2.5 The Lattice of Subgroups of a Group

The join of subgroups H, K is the smallest subgroup which contains both. It is written \(\langle H, K\rangle \).

The lattice of subgroups is a visualization of all subgroups of a group. It places larger groups higher, and draws edges between subgroups if one is a "direct" subgroup of the other (i.e. there is no other subgroup "in-between").

We can use the lattice to help determine certain subgroups. Since for instance, the centralizer, center, normalizer, and other defined structures are subgroups, they must occur in the lattice. Since the lattice is often listed in terms of generators, we can check whether the generator is in the group to determine if the groups are equal.
