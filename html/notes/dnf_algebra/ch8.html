Chapter 8

Euclidean Domains, PIDs, and UFDs

Section 8.1

Euclidean Domains

In this chapter we assume all rings are commutative. A norm on an integral domain R is any function N:R→ℤ+∪{0} with N(0) = 0. If N takes only positive values on non-zero arguments, it is called a positive norm. A ring R is called a Euclidean domain if there is a norm N on R such that for any a, b ∈ R and if b ≠ 0, there are q, r ∈ R with $$ a = qb+r \quad \text{ with } r=0 \text{ or } N(r) < N(b) $$ We call q the quotient and r the remainder.

The decreasing nature of the remainder is that it allows for successively smaller decompositions of the quotient. That is to say it allows the chain of equalities $$ a = q_0b+r_0 $$ $$ b = q_1r_0+r_1 $$ $$ \vdots $$ $$ r_{n-1} = q_{n+1}r_n $$ The first equation decomposes a into, so to speak, blocks of size b with a remainder r. (Of course this is all in analogy with positive integers.) But then we decompose the quotient itself into smaller pieces. In fact, pieces of size r, but then a new remainder is left over. So we continue the process of refinement until no further refinement is possible.

To make explicit the analogy, take 37 and 8. We first find 37 = 8*4 + 5, which you can visualize as four blocks of size 8, followed by a block of 5. But then we decompose the blocks of size 8 each into blocks of size 5. This will allow us to now group all the blocks of size 5, but now each block of size 8 leaves behind a block of 3. So we decompose each block of 5 into a component of size 3, and so on until we are only left with unit-sized blocks. This merely reflects the fact that 37 and 8 have greatest common divisor equal to 1. We will elaborate on this point below, returning back to the ring-theoretical view.

The chain of equations above (or any algorithm which generates them) we call the Euclidean algorithm.

Theorem: Every ideal of a Euclidean domain is prinicpal.

Explanation:

If the ideal is 0 there is nothing to prove. Otherwise to prove that an arbitrary ideal I is principal, intuitively it should be the case that the generator is "small" and can then generate all the big stuff. Since Euclidean domains are equipped with a norm which in some sense measures the size of elements, we set d to be any nonzero element of minimal norm and anticipate that it generates the ideal.

Now let a ∈ I and "use the division algorithm" to show that d generates a. That is to say, invoke the property of the norm that a = dq+r for some q, r ∈ R. The definition of the norm is practically built to tell us that, since d was chosen minimally, we cannot have a nonzero r such that N(r) < N(d). So we must have r=0. But then a = dq and therefore a ∈ (d).

We say that a is a multiple of b if there is some x such that a=bx (assuming b ≠ 0). We also say for the same thing, that b divides a and write for this b | a. An element d is a common divisor of a, b if it divides each. A greatest common divisor will be the greatest element from the set of divisors, where "greatness" is measured by divisibility. That is to say, x is greater than y in this context, if y|x.

It is worth noting that a being a multiple of b, in the language of rings, is the same as being in the ideal generated by b. In fact this is even equivalent to (a) ⊆ (b). Thereby a common divisor is an element which generates an ideal containing both of the other ideals, and a greatest common divisor generates an ideal contained in all of the ideals of all common divisors.

The following theorem's proof reiterates many of these ideas. It is worth emphasizing that the theorem is only a sufficient, and not a necessary, condition.

Theorem: If a ≠ 0 ≠ b generate some principal ideal, i.e. if (a,b) = (d), then d is the GCD of a and b.

Explanation: Certainly a ∈ (d) so that d | a, and likewise for b, so d is a common divisor. If any x is also a common divisor, then x ∈ (a,b) so x ∈ (d) so x | d. Hence, with the meaning we give to "greater than" in this setting, d is greater than x.

Theorem: If R is an integral domain and d ≠ d' generate the same ideal, then d, d' are unit multiples of each other, i.e. there exists a unit u ∈ R such that d = ud'.

Explanation: $$ d = d'x $$ $$ d' = dy $$ $$ d = dyx $$ $$ d(1-yx) = 0 $$ If d = 0 the result would have been immediate, and otherwise yx = 1 so that y and x are units.

Theorem: The last remainder, d=rn, in the Euclidean algorithm is the GCD for a, b (with a≠0≠b), and (d)=(a,b). This last implies Bezout's identity.

Explanation: In the final equation, it is immediate that the last remainder divides the one before it. Given that fact and the equation before it, it is only slightly less immediate that the penultimate remainder divides the one before it. Once this is established it is also quite immediate that the last remainder divides the antepenultimate. Proceeding inductively up the chain we see that the last remainder divides every remainder in the Euclidean algorithm. From this it follows that d divides b and from this it follows that d | a, so d is a common divisor. If any other number, d', is a common divisor, we can in fact work a similar argument but this time working from the first equation to the last, to show that d' | d.

We now have that d is the GCD and from the earlier theorem (d) = (a,b) from which everything else follows quickly.

For an integral domain R we define R̃ as the set of units together with 0. An element u ∈ R⟍R̃ is called a universal side divisor if for all x ∈ R there is some z ∈ R̃ such that u | x-z in R. This is a weakening of the Euclidean condition, since it says that every x may be written as x = qu+z where z=0 or is a unit.

Theorem: Let R be an integral domain which is not a field. If R is a Euclidean domain then there are universal side divisors.

Explanation: Again we take an element of minimal norm (from R⟍R̃) and expect that it may exhibit the object we seek (in this case, a universal side divisor). Again the definition of the norm together with minimality tells us that this element of minimal norm must be a universal side divisor.

Section 8.2

PIDs

A principal ideal domain (PID) is an integral domain in which every ideal is principal. We have already seen in the previous section that every Euclidean domain is a PID. In fact, the proofs of two theorems earlier made no necessary use of Euclideanness--they could just as well be re-stated for PIDs.

Theorem: In a PID if (a,b) = (d) then the GCD of a, b is d, the Bezout identity holds, and d is unique up to multiplication by a unit.

Explanation: Already given.

We've seen that maximal ideals are prime but not necessarily the converse. However, the converse does hold in PIDs.

Theorem: Every nonzero prime ideal in a PID is maximal.

Explanation: If (a) is prime and r ∈ R⟍(a), we show that (a,r) = R. Since R is a PID we must have (a,r) = (p) for some p ∈ R and so a = px for some x ∈ R. Since we assumed (a) was prime, either p ∈ (a) or x ∈ (a).

Now if p ∈ (a) then we can show r ∈ (a) contrary to assumption. This follows since p=ay and r=pz so r=ayz. So since p ∉ (a) then we must have x ∈ (a). Then a = p(ab) so a(1-pb) = 0 and it quickly follows that p is a unit. Hence (p) = R.
