Chapter 7

Rings

Section 7.1

Basics

A ring is an abelian group with a * operation that distributes on the left and right. We call the ring commutative if the * operation is. We say it has a unitary element, 1, if it is an identity for *.

It is interesting to note that a ring wouldn't make sense if + were not abelian, due to the requirement for distributivity. In particular (1+1)(a+b)=a+b+a+b=a+a+b+b which requires a+b=b+a.

The ring is called a division ring if every non-zero element has a multiplicative inverse. If a division ring is commutative we call it a field.

Theorem: 0a = a0 = 0, and (-a)b=a(-b)=-ab, and (-a)(-b)=ab, and if the ring has unity, -a=-1a.

Explanation:

These follow from tricky uses of the additive subgroup together with distribution. 0a = (0+0)a = 0a+0a so now cancel. For the next one (-a)b+ab = (-a+a)b = 0b = 0 making ab the inverse of (-a)b. For the next (-a)(-b)-ab = (-a)(-b)+(-a)b = -a(-b+b) = -a0 = 0. Finally, (-1)a = -(1a)=-a.

A non-zero element is called a zero divisor if there is some non-zero element it multiplies with to zero. It is a unit if it has a multiplicative inverse. Clearly a unit cannot be a zero-divisor since, if u is a unit, then ux = 0 implies x = 0 by multiplying by the inverse.

We can talk about the group under multiplication, if we only consider the units. This is called the group of units.

A commutative ring with unit \(1\ne 0\) is called an integral domain if there are no zero divisors.

Theorem: If a is not a zero divisor then ab=ac implies either a=0 or b=c (a weakened cancellation property, making not use of an inverse).

Explanation: Move to one side, factor, use the not-zero-divisor property.

Corollary: A finite integral domain is a field.

Explanation: because we have the cancellation law above, the map \( x\mapsto ax \) is injective for any a. Hence it is a bijection and therefore sends some x to 1.

A subring is a subgroup closed under multiplication.

A very important example is \( \mathbb Z[\sqrt D] \) where D is a square-free integer. We typically use the field norm \(N(a+b\sqrt D)=|a^2-Db^2|\) because it is multiplicative, i.e. \(N(\alpha\beta)=N(\alpha)N(\beta)\), and because an element is a unit if and only if \(N(\alpha)=1\).

Section 7.2

Polynomial Rings, Matrix Rings, Group Rings

In the example of polynomials we assume a commutative ring with identity for our coefficients. Then the ring of polynomials with coefficients in a ring, R[x], is a ring. Addition is defined in the obvious way, coordinate-wise. Multiplication is defined in the usual way, convoluting the terms. That is to say the coefficient of the xk is \(\sum_{i=0}^k a_ib_{k-i}\). The degree is the largest power of a monomial with non-zero coefficient.

Theorem: The degree of a product is the sum of degrees, if both polynomials are not zero. The only units are the constants which are also units in R. R[x] is an integral domain.

Explanation: The degree claim is clear from the formula for the coefficients of a product. The fact that units in R are units in R[x] is even clearer, but what is challenging is to know that no non-constant polynomial is a unit. But of course, the degree fact just proved makes the demonstration easy. Finally we want to see that there are no zero-divisors, but again the degree formula makes this easy.

Next we define a matrix ring with coordinates in R. These are \(n\times n\) matrices with addition and multiplication defined in the obvious ways.

And finally we define group rings for an arbitrary commutative ring R with identity \(1\ne 0\) and an arbitrary group G. Then the group ring RG is the set of all formal sums $$\sum a_ig_i \quad a_i\in R $$ with addition and multiplication defined in the usual way.

Section 7.3

Ring Homomorphisms and Quotient Rings

A homomorphism is defined as you'd expect, just as with kernel.

Theorem: Kernels and images of ring homomorphisms are subrings.

Explanation: The proof is straight-forward just as in the case of groups.

Naturally we want an analogue to normal subgroups for rings, in order to both understand kernels of homomorphisms as well as to define a quotient ring. We call such a set an ideal, and we understand each coset to be represented under addition. That is to say, if I is an ideal and r any ring element, then the coset corresponding to r is r+I.

Here the analogy with groups breaks down, since ideals will ned to have a property unlike any property of normal subgroups. Notice that in order for the ring multiplication to be define we will require that, for any ideal I, and any two α ≠ β ∈ I, the choice of different representatives shouldn't matter. In particular, this implies that for any two r, s ∈ R, $$ (r+\alpha+I)(s+\beta+I) = rs + I $$

From this we can extract some requirements on how I interacts with ring elements. In particular if s = 0 then rβ+αβ must be an element of I. But of course we will require I to be a subring, so αβ is already in I. So rβ is in I. But this means that I must be closed under arbitrary products and not just closed under products within I alone. That is to say, rI = I.

For non-commutative rings we can distinguish left-ideals and right-ideals, and two-sided ideals.

Theorem: The first four isomorphism theorems hold for rings.

Explanation: The proof is effectively the same as for groups.

Section 7.4

Properties of Ideals

Throughout this section we assume R is a ring with identity 1 ≠ 0. The definition of the ideal generated by A is as you'd expect. A principal ideal is an ideal generated by just one element. A finitely generated ideal is what you'd expect. It is worth noting that an ideal generated by a single element, a, that is to say the ideal (a), is equal to all of the R-multiples of a.

Theorem: An ideal equals the ring if and only if the ideal contains a unit. If R is commutative then it's a field if and only if the only ideals are "stupid".

Explanation: I call something "stupid" if it's unofficially trivial. In this case the ring of just the 0 element, and the entire ring, are stupid ideals.

Since the section assumes the ring has the unitary element, then if I=R then 1 ∈ I. Conversely if u ∈ I is a unit then u-1u = 1 ∈ I, and since 1 ∈ I then for any r ∈ R we have rI = I so r ∈ I.

The second claim is an immediate corollary since fields have only 0 and units, and are commutative.

Corollary: Any nonzero ring homomorphism from a field to a ring is an injection.

Explanation: The kernel of the homomorphism is an ideal and in order to not be R, it must be 0.

An ideal is maximal if there is no ideal between it and the whole ring.

Theorem: Every proper ideal is contained in some maximal ideal.

Explanation: The proof fundamentally rests on Zorn's Lemma, which I'm going to just accept at the same time as I accept this theorem without proof.

Theorem: For commuatitive rings with identity, an ideal is maximal if and only if the quotient ring is a field.

Explanation: If the ring is R and the maximal ideal is M then we use the fact that R/M is a field if and only if its only ideals are stupid. Because M is maximal there is nothing between it and R. Now by the Fourth Isomorphism Theorem (the "lattice" isomorphism theorem), the ideals containing M are in a bijection with all the ideals of R/M. Since the ideals containing M can only be M and R, we're done.

An ideal P is called a prime ideal if P ≠ R and whenever ab ∈ P then at least one of a, b are in P.

Theorem: The prime ideals of ℤ are the ideals pℤ where p is prime, and the 0 ideal.

Explanation: First note that since any ideal is a subring, then any ideal is of the form nℤ. It's easy to check that any such set is an ideal. Now if a, b ∈ R and ab ∈ pℤ, then ab = pk for some k ∈ ℤ. Then p | ab and so by some basic number theory p | a or p | b. Whichever it is, the corresponding element is in pℤ.

Theorem: For commutative rings, an ideal is prime if and only if the quotient is an integral domain.

Explanation: Translate the property of a prime ideal over to a property of the elements in the quotient, using "if and only if" at each step.

Theorem: For commutative rings, maximal ideals are always prime.

Explanation: All fields are integral domains.

Section 7.6

The Chinese Remainder Theorem

We again take all of the rings in this section to be commutative with identity 1 ≠ 0. We define the ring direct product of two rings to be the ring of tuples where both addition and multiplication are defined coordinate-wise. We say that ideals A, B ⊆ R are comaximal if A+B = R. If A = (a), B = (b) then AB = (ab), which can be generalized in a few ways.

The Chinese Remainder Theorem: Let A1, A2, ..., Ak be ideals in R. The map R → R/A1 × R/A2 × ... × R/Ak defined by r ↦ (r+A1, r+A2, ..., r+Ak) is a ring homomorphism with kernel ⋂Ai. If all of the distinct i ≠ j correspond to comaximal Ai, Aj, then the map is surjective and the kernel equals ∏Ai.

Explanation: We prove the case for k = 2 and every other case follows from a simple inductive argument. Since the map is just the natural projection onto the quotient groups, it is clearly a homomorphism. Any element is in the kernel if and only if it's in both A and B.

Now assume that A, B are comaximal and let a ∈ A, b ∈ B such that a+b = 1. Then φ(a) = (0,1), φ(b) = (1,0). We now show that φ is surjective by considering an arbitrary (r+A, s+B). Then φ(as+br) = (r+A, s+B).

Finally we show that A∩B = AB. It is always true that AB ⊆ A∩B. Now let c ∈ A∩B and using the same a+b = 1 as above it follows that c = ca+cb ∈ AB. Note that the inclusion holds because the product of rings is defined by containing all sums of all products.

The familiar version of the Chinese Remainder Theorem comes from first proving ℤ/mnℤ ≅ (ℤ/mℤ)×(ℤ/nℤ), then using the above theorem to infer that the isomorphism still holds for the corresponding groups of units.
