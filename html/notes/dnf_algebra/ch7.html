Chapter 7

Rings

7.1 Basics

A ring is an abelian group with a * operation that distributes on the left and right. We call the ring commutative if the * operation is. We say it has a unitary element, 1, if it is an identity for *.

It is interesting to note that a ring wouldn't make sense if + were not abelian, due to the requirement for distributivity. In particular (1+1)(a+b)=a+b+a+b=a+a+b+b which requires a+b=b+a.

The ring is called a division ring if every non-zero element has a multiplicative inverse. If a division ring is commutative we call it a field.

Theorem: 0a = a0 = 0, and (-a)b=a(-b)=-ab, and (-a)(-b)=ab, and if the ring has unity, -a=-1a.

Explanation:

These follow from tricky uses of the additive subgroup together with distribution. 0a = (0+0)a = 0a+0a so now cancel. For the next one (-a)b+ab = (-a+a)b = 0b = 0 making ab the inverse of (-a)b. For the next (-a)(-b)-ab = (-a)(-b)+(-a)b = -a(-b+b) = -a0 = 0. Finally, (-1)a = -(1a)=-a.

A non-zero element is called a zero divisor if there is some non-zero element it multiplies with to zero. It is a unit if it has a multiplicative inverse. Clearly a unit cannot be a zero-divisor since, if u is a unit, then ux = 0 implies x = 0 by multiplying by the inverse.

We can talk about the group under multiplication, if we only consider the units. This is called the group of units.

A commutative ring with unit \(1\ne 0\) is called an integral domain if there are no zero divisors.

Theorem: If a is not a zero divisor then ab=ac implies either a=0 or b=c (a weakened cancellation property, making not use of an inverse).

Explanation: Move to one side, factor, use the not-zero-divisor property.

Corollary: A finite integral domain is a field.

Explanation: because we have the cancellation law above, the map \( x\mapsto ax \) is injective for any a. Hence it is a bijection and therefore sends some x to 1.

A subring is a subgroup closed under multiplication.

A very important example is \( \mathbb Z[\sqrt D] \) where D is a square-free integer. We typically use the field norm \(N(a+b\sqrt D)=|a^2-Db^2|\) because it is multiplicative, i.e. \(N(\alpha\beta)=N(\alpha)N(\beta)\), and because an element is a unit if and only if \(N(\alpha)=1\).

Section 7.2

Polynomial Rings, Matrix Rings, Group Rings

In the example of polynomials we assume a commutative ring with identity for our coefficients. Then the ring of polynomials with coefficients in a ring, R[x], is a ring. Addition is defined in the obvious way, coordinate-wise. Multiplication is defined in the usual way, convoluting the terms. That is to say the coefficient of the xk is \(\sum_{i=0}^k a_ib_{k-i}\). The degree is the largest power of a monomial with non-zero coefficient.

Theorem: The degree of a product is the sum of degrees, if both polynomials are not zero. The only units are the constants which are also units in R. R[x] is an integral domain.

Explanation: The degree claim is clear from the formula for the coefficients of a product. The fact that units in R are units in R[x] is even clearer, but what is challenging is to know that no non-constant polynomial is a unit. But of course, the degree fact just proved makes the demonstration easy. Finally we want to see that there are no zero-divisors, but again the degree formula makes this easy.

Next we define a matrix ring with coordinates in R. These are \(n\times n\) matrices with addition and multiplication defined in the obvious ways.

And finally we define group rings for an arbitrary commutative ring R with identity \(1\ne 0\) and an arbitrary group G. Then the group ring RG is the set of all formal sums $$\sum a_ig_i \quad a_i\in R $$ with addition and multiplication defined in the usual way.
