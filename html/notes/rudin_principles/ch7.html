Section 1

Discussion of the Problem

We would like to know under what conditions we can inter-change limits, limt→xlimn→∞fn(t) = limn→∞limt→xfn(t). We can see from the example \(s_{m,n} = \frac{m}{m+n}\) that if you take the limit of each variable going to infinity, the order matters.

Section 2

Uniform Convergence

We define a sequence {fn} of functions to be uniformly convergent on a set E if for every ε ∈ ℝ+ there is an integer N such that for all n ≥ N $$ |f_n(x) - f(x)| \leq \varepsilon $$ for all x ∈ E. There is a Cauchy criterion for this convergence.

The Cauchy Criterion Theorem: A sequence {fn} converges uniformly if and only if there is an N such that for all m,n ≥ N and x ∈ E $$ |f_n(x)-f_m(x)| \leq \varepsilon $$

Explanation: If the sequence uniformly converges, use the usual wiggle in trick with the triangle inequality. If the sequence is Cauchy note that at each point the sequence of points is Cauchy and therefore converges. Hence we so define a function, and show that the sequence converges to that function.

To show this, set ε ∈ ℝ+ and find the corresponding N. Since the Cauchy criterion holds for every choice of m, then take the limit as m→∞. This leaves you with f(x) in place of fm(x) in the statement. This then becomes the statement of uniform convergence.

Theorem: Suppose the sequence point-wise converges to a function. If and only if we also have that the limit of the diameters goes to zero, limn→∞supx∈E|fn(x)-f(x)| = 0, then the sequence uniformly converges.

Explanation: The proof is immediate due to the definition of uniform convergence.

Theorem: If |fn| ≤ Mn, and if ∑Mn < ∞ then ∑fn < ∞.

Explanation: Show the Cauchy condition for the sequence of functions, which involves showing the sum from m to n becomes small. Use the triangle inequality, then the bounds, then the fact that the series of bounds has the Cauchy property.

Section 3

Uniform Convergence and Continuity

Theorem: If \(f_n\overset{u}{\to} f\) on a set E, and if x is a limit point of E, then $$ \lim_{t\to x}\lim_{n\to\infty}f_n(t) = \lim_{n\to\infty}\lim_{t\to x}f_n(t) $$

Explanation: For each n we define limt→xfn(t) = yn(x). That stated theorem is equivalent to showing limt→xf(t) = limn→∞yn(x).
