Finite, Countable, and Uncountable Sets

I don't know why this section is in a chapter on topology, it could have gone in a section on mere pre-requisites. We define familiar ideas about functions, domain, range, image, into, onto, inverse, one-to-one, corerspondence, cardinality, equivalence relation, finite, infinite, countable, and uncountable. One oddity is that finite sets are not defined in such a way as to be countable.

Later we define sequence, union, intersection, and disjiont.

Along the way we prove the following.

Theorem: Every infinite subset of a countable set is countable.

Theorem: The usual set laws of commutativity, associativity, and distributivity are true.

Theorem: Countable unions of countable sets are countable.

Theorem: The cartesian product of countable sets is countable.

Theorem: The rationals are countable.

Theorem: The set of all sequences of 0s and 1s is uncountable.

Section 2

Metric Spaces

Again I'm not sure why this is in topology per se, since topologies don't always have distance metrics. But here we define a distance metric by positivity, zero if and only if the points are equal, commutativity, and the triangle inequality. On the real line we define the segment (a, b) to be the set of all numbers a < x < b. By the interval [a,b] we mean all \(a \leq x\leq b\). A k-cell is all \(\vec x \in \mathbb R^k\) such that for each coordinate there is an interval the coordinate must be in. The open ball centered at \(\vec x\in \mathbb R^k\) with radius r is the set of all \(\vec y\in \mathbb R^k\) such that \(|\vec x - \vec y | < r\). A set E is convex if for every \(\vec x,\vec y\in \mathbb R^k\) and every \(0 < \lambda < 1\) we have $$ \lambda \vec x + (1-\lambda)\vec y \in E $$

A neighborhood around the point \(p\in X\) of radius r is denoted by \(N_r(p)\) and is the set of all points within distance r of p. A limit point of a set E is any point such that a neighborhood always contains some other part of E. An isolated point of E is a point in E that is not a limit point. E is closed if it contains all its limit points. A point is an interior point if there is some neighborhood around it which is entirely in E. E is open if every point is interior. E is perfect if a point is in it if and only if it's a limit point. E is dense in X if every point in X is either in E or a limit point of E.

Theorem: Neighborhoods are open sets.

Explanation: Take any neighborhood and show every point in it is an interior point. To do that, fit a neighborhood around the new point which fits inside the first neighborhood. To do that you need to pick the right radius. If you draw a picture you can convince yourself that the right radius is the first radius minus the distance between the points.

(You can draw the picture pretty easily in two dimensions, although it works in one or three. This problem doesn't assume we're in a Euclidean space so the picture is merely meant to motivate a choice of radius, but is not a proof. The proof uses the triangle inequality, after you've made this well-motivated guess about the choice of radius.)

Theorem: If p is a limit point of E then E contains infinitely many points.

Explanation: The picture makes this obvious, and it's not too much work to turn the picture into a proof. Draw any blob of a set E and pick a limit point. Best to draw it on the edge of the set E since that's where the most interesting limit points live. Put any neighborhood around p. By definition as a limit point, we get some distinct other point in E, call it \(q_1\).

Now we generate a \(q_2\). How? We need it to be distinct from \(q_1\), and we can guarantee this if we use \(d(p,q_1)\) as the radius of a new neighborhood around p. Get a point from inside there, call it \(q_2\). The radius puts \(q_1\) outside this new neighborhood so the \(q_2\ne q_1\) and we have a genuinely new point.

Generate \(q_3\) with a neighborhood of radius \(d(p,q_2)\), and generate \(q_4\) with a neighborhood of radius \(d(p,q_3)\) and so on.

Corollary: A finite set has no limit points.

Explanation: Notice that you can prove this directly without too much difficulty. Still, it's a nice corollary from the previous theorem. In fact, it is just the contrapositive of the previous theorem.

Theorem: De Morgan's law is true.

Why is this here? This is a set theory fact and should be in some section with set theory facts.

Theorem: A set is open if and only if its complement is closed.

Explanation: We prove each direction separately. First suppose E is open, we want to prove \(E^c\) contains all its limit points, so say that x is one. This easily entails by definition that x is not interior to E, but E is made up only of interior points. So \(x\not\in E\) so \(x\in E^c\).

Next suppose \(E^c\) is closed, and show that any point in E is interior. By being in E then x cannot be a limit point of \(E^c\), and that gives you what you need to put a neighborhood around x which is entirely in E.

Theorem: A set is clsoed if and only if its complement is open.

Explanation: This is a very nice and easy corollary to the previous theorem. If F is closed, then the complement of \(F^c\) is closed and therefore \(F^c\) is open. Do similarly to go in the other direction.

Theorem: Unions of open sets are open. Finite intersections of open sets are open. Intersections of closed sets are closed. Finite unions of closed sets are closed.

Explanation: The easiest claim in here is the first one. If you pick a point in the union it must have come from one of the open sets, and now put a neighborhood fitting inside that set.

Having proved the first claim, the next easiest is that intersections of closed sets are closed. Construct the closed set as the complement of an open set, which by De Morgans is now a union of open sets, and so on.

For finite intersections of open sets, finiteness is the key: Any point in the intersection is in all the open sets. For each of them, take a neighborhood. Then take the least radius of those neighborhoods (which exists because there are only finitely many of them). That will fit in all of them.

Unsurprisingly, we can again leverage complements and De Morgan's to prove the claim about closed sets from the fact about open sets.

The closure of a set E is the set unioned with its limit points. We denote the set of limit points by E' and the closure by \(\overline E\).

Theorem: The closure of any set is closed. A set is closed if and only if it equals its closure. \(\overline E\) is contained in any closed set that contains E.

Explanation: We start with showing that the closure of any set is closed. Rudin proves that the complement is open, which is nice. But I go for a direct proof: Take any limit point of \(\overline E\). Intuitively it's either a limit point of E or a limit point of E'. If it's a limit point of E we're done because it's automatically in E'. But if it's a limit point of E' then we still need to do some work to show that this is a limit point of E. Intuitively that should be true because if anything is infinitely to E' and E' is infinitely close to E, then the point should also be infinitely close to E. But all of this is just getting some intuitions--next we need a plan for a rigorous proof.

Let x be a limit point of \(\overline E\). We are trying to show it's a limit point of E, so let's put a neighborhood \(N_r(x)\) around it. We want to show that \(N_r(x)\) must have a point other than x which is in E.

Since there is a point \(y\in E\cup E'\) in the neighborhood where \(x\ne y\) then if \(y\in E\) we're done. If \(y\in E'\) then ... well, we can put a neighborhood around y which must intersect E. If we can choose this neighborhood small enough so that the whole thing fits inside the original neighborhood, then we're done. But to do this is just like how we showed that all neighborhoods are open. We draw a picture and let it motivate a choice of radius.

For the claim that a set is closed just in case it equals its closure: One direction is trivial. If \(E=\overline E\) then since we already proved \(\overline E\) is closed, automatically E is closed. The other direction is actually also trivial because if E is closed then it contains all its limit points, so \(E'\subseteq E\) and therefore \(E = E\cup E' = \overline E\).

That \(\overline E\) is contained in any closed set that contains E: Let F be any closed set with \(E\subseteq F\). We will have \(\overline E \subseteq F\) if we can just show \(E'\subseteq F\). So if x is any limit point we need to show it's in F. Clearly we need to appeal to the closedness of F, but you quickly realize that F containing its limit points means it also contains the limit points of E.

Theorem: The sup of a nonempty bounded set is always in the closure.

Explanation: This naturally breaks into two cases, depending on whether the sup is in the set or not. If it is, the case is trivial. If it's not then we prove it's a limit point. We take a neighborhood of the sup and in particular look at the part that's below the sup. Since this region is less than the sup, none of it is an upper-bound and so there must exist some element of the set above. Because the sup was not in the set, this element that we've found cannot be the sup itself. Hence we found a distinct point in the neighborhood.

It sometimes happens that we study a space, X, but then want to study some subspace \(Y\subset X\). Sets that were open in X may no longer be open if we regard it as now living just in Y. For instance, [0,1) is actually an open set if the universal set is \([0,\infty)\). Hence it really is illegitimate to say that a set is open--we can only say that it is open "relative to" the metric space we consider it to live in. We rarely need to worry about this subtlety but when we do, we have the following helpful theorem.

Theorem: Suppose \(Y\subseteq X\). A set E is open relative to Y if and only if \(E=Y\cap G\) for some set G which is open relative to X.

Explanation: Use the example of \(E = [0,1)\) being open relative to \(Y = [0,\infty)\) for motivation. We can see that \(E = Y\cap G\) with \(G = (-1,1)\). The over-hang on the left side gets chopped off by Y so it doesn't matter. We could have used (-2,1) and it would have worked just as well for a choice of G.

Another way to see this is to imagine the point 0.1 in [0,1) and the neighborhood of radius 0.2. If we are living in Y when we do this, the neighborhood is [0,0.3). But if we did this when living in X the neighborhood would have been (-0.1, 0.3). But the neighborhood from X is still a good neighborhood in Y if we just intersect it with Y first.

This gives an indication of how to prove the left-to-right direction. We build the set G by taking points in E and neighborhoods which fit within it while living in Y. We use the same radius, but now construct the neighborhood while living in X. Unioning all these neighborhoods constructs G. Intersecting with Y lops off the part that gets us back down to E.

For the right-to-left: This just falls right out of definitions.

Section 3

Compact Sets

An open cover of a set is any collection of open sets which covers. "Covers" means that the set is contained in the union of the collection. A set is compact if every open cover has a finite subcover.

Theorem: Compactness survives the restriction to a subspace.

Explanation: More formally, the theorem claims that if \(K\subseteq Y\subset X\) then K is compact relative to X if and only if it's compact relative to Y. To to left-to-right: take an open cover relative to Y and then use the fact that open sets relative to Y have an open set G relative to X.

The right-to-left is similar.

Theorem: Compactness entails closedness.

Explanation: In general when proving openness or closedness we have an option. To prove that a compact set is closed, we can either directly prove it contains its limit points or we can prove its complement is open. In this case it's hard for me to imagine linking the idea of a limit point to an open cover. But to show the complement is open, we can can use a point outside the set to build an open cover. So let's chase down that line of thinking.

We pick a point in the complement, \(p\in K^c\) and try to build an open set V around it that is guaranteed to be entirely in the complement, i.e. \(V\subseteq K^c\). To do this, we take every point in the compact set, \(q\in K\), and build two open sets, one around p and one around q. These two open sets, call them \(V_q\) and \(W_q\), split the difference the distance from p to q. Intuitively, the Vs will cover p and eventually become our V by taking intersections. And Ws will become the open cover from which we select a finite subcover. With these insights in mind, the rest is chasing down the details.

Theorem: Closed subsets of compact sets are compact.

Explanation: Clearly somehow the closed set inside has to leverage the compact superset. We take an open cover of the closed set and don't yet know why there should be a finite subcover, since this isn't necessarily an open cover of the superset. But here's the kicker: We can add one more open set to get an open cover of the superset. It's the complement of the closed set.

Corollary: The intersection of a closed set and a compact set is compact.

Explanation: The compact set is closed. Intersections of closed sets are closed. Closed subsets of compact sets are compact.

Theorem: For compact sets, finitely intersecting is sufficient for intersecting.

Expalantion: If \(\{K_\alpha\}\) is any collection of compact sets, the theorem claims that if every finite \(\{K_{\alpha_1}, ..., K_{\alpha_n}\}\) intersects then so does all of \(\{K_\alpha\}\).

The theorem sounds so similar to the definition of compactness, execpt somehow stated in an opposite form. In the definition, every infinite union of open sets has a finite subset. Here every finite intersection of compact sets entails an intersection of all compact sets ... stated very roughly, in order to highlight the parallels. It sure feels like somehow one is the same as the other just using De Morgan's to flip open sets to closed and unions to intersections.

To argue by contradiction we assume that the intersection is empty. That must mean that if we pick any set, call it \(K_1\in \{K_\alpha\}\), each of the elements of \(K_1\) must be lacking in some other set. Well that means that for each element, there exists a \(K_\alpha^c\) which contains it and is open. This is shaping up to be an open cover, from which we select a finite subcover. From which we can get a contradiction with the fininte intersecting property.

Corollary: Any decreasing chain of non-empty compact sets has an intersection.

Explanation: Because each set in the chain is non-empty, any finite intersection is just equal to the last set, which is non-empty. That means this has the finite intersection property, so apply the previous result.

Theorem: Infinite subsets of compact sets have at least one limit point in the compact set.

Explanation: For proof by contradiction, if every point is not a limit point then this entails the existence of a neighborhood around each point. This infinite collection covers but no finite subcover does.

Theorem: A descending chain of intervals of real numbers has non-empty intersection.

Explanation: The similarity to the corollary above is striking. In fact, in a few theorems down we'll learn that every k-cell is compact, and in this case (i.e. for \(\mathbb R^1\)) a k-cell just is an interval. So this really is a special case of the earlier corollary.

Still Rudin gives a direct proof and maybe it's worth understanding that. In fact the trick of the proof is semi-famous: we investigate the sup of the left endpoints of the interval. This must be greater than all left endpoints and less than all right endpoints.

Theorem: A descending chain of k-cells also has non-empty intersection.

Explanation: Perform the same basic argument as before, but coordinate-wise. Or, again, note that we will soon learn that k-cells are compact, and we already know that descending chains of non-empty compact sets intersect.

Theorem: k-cells are compact.

Explanation: For contradiction we suppose that it is not compact, so that for some open cover, there is no finite subcover. The fundamental intuition here is that, if you take any finite subcover it must not cover some part of the k-cell. In fact, if we evenly sub-divide the k-cell, some one of these subdivisions must not be finitely subcoverable (for if they all could be, then the whole of them could be). But if that sub-k-cell cannot be finitely subcovered, then we can repeat the process and get an even smaller sub-k-cell, and so on.

In this way we generate a descending chain of k-cells. We apply the previous result to find a point in their intersection. This point must be inside some open set in the open cover. From there we are in a good position to argue that some one of the k-cells which we showed is not finitely subcovered, is also covered by this one open set.

The Heine-Borel Theorem (and more): Compact is equivalent to closed and bounded in \(\mathbb R^k\), is equivalent to every infinite subset has a limit point in the set.

Explanation: Even in general metric spaces, compactness is not equivalent to being closed and bounded. But in \(\mathbb R^k\) they are equivalent and we'll start by showing this. First, we already know compactness entails closedness. All that remains for this direction is to show that compactness entails boundedness. But for contradiction, if a set were both compact and unbounded, we could find an infinite sequence of points with no limit point, contradicting an earlier theorem.

That any closed and bounded set must be compact: Suppose K is any closed, bounded subset of \(\mathbb R^k\). Since it's bounded it fits in a k-cell, and closed subsets of compact sets are compact.

That compactness entails that every infinite subset has a limit point in the set: This is the content of a theorem earlier.

If every infinite subset has a limit point in the set then it's closed and bounded: Let E be any set with the following property: If \(F\subseteq E\) and F is infinite, then there is a limit point of F in E. The theorem claims that therefore E is closed and bounded. We do both by contradiction. Bounded is easier so we start with that.

Bounded: If the set is unbounded then we make a sequence of points going to infinity, and the limit of this sequence clearly isn't in the set.

Closed: If it's not closed then there is a point which is a limit point but not in the set. Then construct a sequence converging to the point. This sequence cannot have any other limit point, easy to prove that. And so on.

Weierstrass's Theorem: Every bounded infinite subset of \(\mathbb R^k\) has a limit point in \(\mathbb R^k\).

Explanation: Being bounded, it fits in a k-cell. Use the theorem above on the k-cell.

Section 4

Perfect Sets

Theorem: Nonempty perfect sets are always uncountable.

Explanation: Since some point is a limit point, then it has infinitely many points. To show that it's uncountable, we assume for contradiction it's countable. So of course assume \(\vec x_1, \vec x_2, ...\) is an enumeration. Start by finding a neighborhood \(V_1\) around \(\vec x_1\). What contradiction are we driving toward? We can build a descending chain with no intersection. To that end, construct a neighborhood that excludes \(\vec x_1\) and fits inside \(V_1\). Then take its closure. And so on, constructing neighborhoods that avoid all the other points.

How do we know we can get this \(V_2\)? Since \(\vec x_1\) is a limit point we can find an \(\vec x_i\) inside \(V_1\) and fit an obvious ball around that.

Theorem: Every interval of positive length is uncountable.

Explanation: Take the closure of any interval of positive length, it's perfect.

The Cantor set is constructed by \(E_1 = \left[0,\frac 1 3\right]\cup \left[\frac 2 3, 1\right]\) and then \(E_2\) removes the middle third of the left and right parts of \(E_1\), and so on with \(E_3, E_4, ...\) Then the Cantor set is \(\displaystyle P = \bigcap_{n=1}^\infty E_n\). Clearly P is bounded, and arbitrary intersections of closed sets are closed, so P is compact.

Rudin claims that showing P is perfect is implied by showing it has no isolated points. This is because limit points are precisely those points which are not isolated points. To show that P has no isolated points, if \(x\in P\) then \(x\in E_n\) for some n. Then \(x\in I_n\) where \(I_n\) is the interval of \(E_n\) which contains x. For any neighborhood of x we can find an n large enough such that \(I_n\) fits inside, and then take the endpoints of \(I_n\).

Section 5

Connected Sets

Sets A and B are separated if both \(A\cap\overline B\) and \(\overline A \cap B\) are empty. A set E is connected if it is not the union of two separated sets.

Theorem: A set of real numbers is connected if and only if it contains every intermediate point.

Explanation: If it did not contain some intermediate point then this could be used as a "point of separation". Take the part of it to the left of the absent point, the part to the right, and show these are separated sets.

On the other hand, if the set is separated, then we can take any two points and construct a gap between them. In particular, if the set is separated into A and B then pick \(x\in A\) and \(y\in B\). Without loss of generality, we suppose x < y. What is the gap between them? \(z = \sup (A\cap [x,y])\) is a good start, especially if we luck out and have \(z \not \in A\). The hard part is if \(z\in A\).
